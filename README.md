# Gym-Tracker-App

A REST API for tracking gym workouts built with Spring Boot and PostgreSql.

## Description
Crud workouts, exercises and edit exercises in workouts

## Tech Stack

 - Java 21
 - Spring Boot 3
 - PostgreSql
 - Docker
 - Swagger UI (for now)


## Prerequisities

 - Java 21
 - Docker
### Db startup
```bash
docker compose up -d
```

### Run the application
Open the project and run `GymTrackerApplication.java`

### API Documentation       
```
http://localhost:8080/swagger-ui/index.html
```
